單、雙冒號語法

最近學python被單、雙冒號搞得好混亂zzzz
做一下語法筆記，幫助自己理解

[ i : j ] 單冒號

i 從第幾個index開始讀取
j 如果是正整數，就是從 i 讀到 j-1；如果是負數，就是從 i 讀到 len -j 的位置

[ i : j : k ] 雙冒號

  • i 從第幾個index開始讀
  • j 到j-1個index結束
  • k 表示讀取的方向，負號就是反向讀取，反之；數字則表示每幾個index讀取一次

Example:

  • [2::-2]
  • [::-1] 將排列順序倒過來
